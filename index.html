EnConvo Settings User Guide v3

A full-scale local manual for EnConvo Settings. This version starts from the live app model: global provider defaults first, then agent-level overrides, then each settings family and the Mavis agent proof map.

Configuration model.

Global Providers Settings define app-wide defaults for model, voice, search, generation, fetch, and OCR providers. New agents inherit those defaults. Agent List pages are per-agent override surfaces, so an agent can use a different model, credential provider, voice, tool set, memory behavior, or run mode.

Screenshot evidence.

Every first-level settings page is backed by a local screenshot. The Agent Configuration (Mavis) example also includes stitched full-scroll composites and setup-cycle screenshots for Anthropic ApiKey and OAuth2 credential branches.
