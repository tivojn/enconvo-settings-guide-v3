EnConvo Settings User Guide v3

A full-scale manual for EnConvo Settings. Start with global provider defaults, then use agent-level settings when a specific agent needs its own model, tools, voice, memory, or runtime behavior.

Configuration model.

Global Providers Settings define app-wide defaults for model, voice, search, generation, fetch, and OCR providers. New agents inherit those defaults. Agent List pages are per-agent override surfaces, so an agent can use a different model, credential provider, voice, tool set, memory behavior, or run mode.

Visual reference.

Each settings page includes a screenshot so you can match the instructions to the EnConvo interface. The AI Model deep dive shows global provider setup examples for EnConvo Cloud Plan and OpenAI. The Agent Configuration (Mavis) example shows per-agent Anthropic ApiKey and OAuth2 credential flows.
